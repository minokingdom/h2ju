# Project Overview

## 1. 프로젝트 목적
이 프로젝트는 **남동구의원 하현주 예비후보**의 온라인 선거 활동을 지원하기 위한 모바일 최적화(웹자보) 싱글 페이지 애플리케이션(SPA)입니다.

## 2. 주요 대상 (Target Audience)
- 인천 남동구 스마트폰 사용자 및 유권자
- 소셜 미디어 플랫폼(카카오톡, 페이스북 등)에서 링크를 클릭해 유입되는 지지자

## 3. 핵심 방향성
- **슈퍼 개인 / 에이전틱 코딩 반영**: 바이브 코딩(Vibe Coding)을 위해 AI가 전체 구조를 망각하지 않게 BKIT를 지속적으로 참조합니다.
- **더불어민주당 아이덴티티**: 전체적으로 더불어민주당 공식 색상인 '민주블루' 톤을 유지하여 신뢰감과 브랜드 일관성을 제공합니다.
- **모바일 퍼스트(Mobile-First)**: 가독성 높은 폰트와 원컬럼/투컬럼 전환형 하이브리드 리스폰시브 레이아웃을 사용합니다.
- **다이내믹 인터랙션**: 모든 컴포넌트는 타이밍 스크롤 애니메이션(`animate-on-scroll`)이 적용되며 자연스러운 전환 및 호버 효과를 지닙니다.

## 4. 파일 구성 요약
- `index.html`: 콘텐츠 뼈대 및 섹션별 레이아웃 정의 영역
- `style.css`: 시각적 아이덴티티, 미디어 쿼리, 반응형 그리드 및 애니메이션 속성 정의
- `main.js`: 네비게이션 스크롤 이동, 인터랙션 Observer 처리 및 후원하기/폼 제출 등의 동작 제어
